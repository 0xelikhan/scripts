# Scripts



| Tool | What it does |
|------|-------------|
| `alert-enricher/` | Takes a Wazuh alert, queries VirusTotal + GeoIP, outputs an incident report |
| `ioc-extractor/` | Parses logs and alerts — pulls IPs, domains, hashes, URLs, CVEs |
| `coverage-reporter/` | Reads Sigma rules, generates a MITRE ATT&CK coverage matrix |
| `lab-health/` | Checks every lab VM and service, reports up/down status |
